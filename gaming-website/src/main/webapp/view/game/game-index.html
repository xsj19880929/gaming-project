LPL2016英雄联盟职业联赛春季赛

举办方：银川圣地国际游戏投资有限公司

比赛区域：全球

服务器：中国

比赛类型：线下赛

开始时间：2016年1月14日

结束时间：2016年4月28日

《英雄联盟》职业联赛（LOL Professional League）是由腾讯举办的中国LOL赛事体系，每年由春季赛和夏季赛组成，每季赛事又分为 常规赛和季后赛。《英雄联盟》职业联赛第五赛季世界总决赛，将在欧洲四座城市进行。参加战队有16只，包括中国大陆、韩国、北美、欧洲各3个名额，台港澳赛区的2个名额，其他地区两个名额。
